## Virtual Environment (virtual muhit)

### Virtural muhit yaratish

```bash
python -m venv .venv
```

### Virtual muhitni ishga tushirish

linux / mac
```bash
source .venv/bin/activate
```

windows
```bash
source .venv/Scripts/activate
```

## Pip (Package manager)

pip install <package_nomi>
